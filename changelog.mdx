## Latest Updates

<Callout kind="info">
  Check this changelog regularly to stay updated on Dezan's new features, bug fixes, and improvements. Each release includes detailed notes on changes that affect your documentation workflows.
</Callout>

Dezan releases follow semantic versioning. Major versions introduce breaking changes, minor versions add features, and patch versions fix bugs.

## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for shared documentation editing
- Introduced advanced search with full-text indexing and filters

## Improvements

- Enhanced performance for large documentation sites with lazy loading
- Improved UI responsiveness on mobile devices

## Bug Fixes

- Fixed markdown rendering issues with nested lists
- Resolved authentication token refresh problems

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated version history with diff viewer for pages
- Added export options for PDF and HTML formats

## Bug Fixes

- Corrected image upload failures in certain browsers
- Fixed sidebar navigation collapse on resize

## Deprecations

- Legacy shortcodes API deprecated; migrate to new components

</Update>

## Upgrade Instructions

Follow these steps to upgrade Dezan to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your documentation repository.

  </Step>
  <Step title="Update Package" icon="download">

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install dezan@latest
    ````

    ````bash
    yarn add dezan@latest
    ````
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute the migration command:

    ````bash
    dezan migrate
    ````

  </Step>
  <Step title="Restart Server" icon="refresh-cw">

    Restart your Dezan instance to apply changes.

  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, review the [migration guide](/introduction#upgrading) for any required manual steps.
</Callout>
