## Sobre esta base

Esta documentacao foi construida a partir de:

- **Codigo-fonte ABAP** baixado via SAP ADT REST API e catalogado (234+ artefatos TXS)

- **Treinamento presencial** de 4 dias sobre Tax Service

- **Analise tecnica** de classes, interfaces e estruturas do pipeline TXS

- **Verificacao contra codigo-fonte** — todos os valores tecnicos (condition types, constantes, nomes de tabelas) foram validados contra o codigo real

<Callout kind="info" collapsed="false">
  Todos os valores tecnicos desta base foram verificados contra o codigo-fonte ABAP. Quando um valor nao pode ser confirmado, ele e marcado com aviso explicito.
</Callout>
