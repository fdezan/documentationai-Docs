## Overview

Dezan supports seamless integrations with popular third-party services, enabling you to automate workflows, sync data, and embed resources directly into your projects. Use built-in app connections, webhooks for real-time events, custom API calls, or embeds to connect Dezan with your existing tools.

<Callout kind="tip">
Review your integration's security requirements before connecting. Always use HTTPS endpoints and validate incoming payloads.
</Callout>

## Setting Up Webhooks

Webhooks allow Dezan to push real-time events to your server. Follow these steps to configure one.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to your Dezan dashboard, go to Settings > Integrations, and select Webhooks. Click "New Webhook".
  </Step>
  <Step title="Configure Endpoint" icon="link">
    Enter your endpoint URL, such as `https://your-webhook-url.com/dezan-events`. Select events like `project.updated` or `user.invited`.
  </Step>
  <Step title="Verify and Test" icon="check-circle">
    Dezan sends a test payload. Verify receipt on your server, then save.
  </Step>
</Steps>

Handle incoming webhooks securely. Here's an example in multiple languages:

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const app = express();
  app.use(express.json());

  app.post('/dezan-events', (req, res) => {
    const event = req.body;
    console.log('Event received:', event.type, event.data);
    // Process event: e.g., notify team, update database
    res.status(200).json({ received: true });
  });

  app.listen(3000);
  ```
  ```python
  from flask import Flask, request, jsonify

  app = Flask(__name__)

  @app.route('/dezan-events', methods=['POST'])
  def handle_webhook():
      event = request.json
      print(f"Event received: {event['type']} {event['data']}")
      # Process event
      return jsonify({'received': True}), 200

  if __name__ == '__main__':
      app.run(port=3000)
  ```
</CodeGroup>

## Custom API Integrations

Build deeper integrations using Dezan's REST API at `https://api.example.com/v1`.

<ParamField path="projectId" param-type="string" required="true">
  Unique project identifier from Dezan.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

### List Projects

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X GET https://api.example.com/v1/projects \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/projects', {
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
  });
  const projects = await response.json();
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "projects": [
      {
        "id": "proj_123",
        "name": "My Project",
        "status": "active"
      }
    ]
  }
  ```
</Response>

## Embedding External Resources

Embed third-party content like charts or forms into Dezan pages.

<Tabs>
  <Tab title="Google Forms" icon="file-text">
    Use iframes for surveys:

````html
<iframe
  src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform"
  width="100%"
  height="600"
  frameborder="0">
</iframe>
````

  </Tab>
  <Tab title="Chart Embed" icon="bar-chart-3">
    Embed interactive charts:

````html
<iframe
  src="https://charts.example.com/embed/CHART_ID"
  width="800"
  height="400">
</iframe>
````

  </Tab>
</Tabs>

<Callout kind="info">
Test embeds in a staging environment to ensure compatibility with Dezan's rendering engine.
</Callout>
