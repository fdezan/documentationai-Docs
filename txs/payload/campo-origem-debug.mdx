```
2. BUILD_HEADER        -> id, date, saleor_purchase, gross_or_net, currency, company_id
                          BR: + operation_nature_code

5. BUILD_PARTY         -> CL_TXS_PARTY (tax registrations)
6. BUILD_BP_EXEMPTION  -> CL_TXS_BP_EXEMPTIONS (vazio no BR)
7. APPLY_CUSTOM_LOGIC  -> BAdI BD_GSLOG_EXT_TAX_CALC_PAYLOAD::EXTEND_PAYLOAD
```
